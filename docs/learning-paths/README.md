## Overview

This directory contains structured learning paths for developers joining the Second Brain OS project. Each path is organized into 5 modules with learning objectives, estimated time, reading materials, and practice exercises.

| Learning Path | File | Modules | Est. Time | Prerequisites |
|---|---|---|---|---|
| Frontend Development | `frontend-dev.md` | 5 | 3-4 days | TypeScript, React basics |
| Backend Development | `backend-dev.md` | 5 | 3-4 days | Python, REST API basics |
| AI/Agent Development | `ai-agent-dev.md` | 5 | 4-5 days | Python, LLM concepts |

## Recommended Order

For full-stack developers, we recommend:

1. **Frontend Development** — Start here to understand the UI layer, component system, and how users interact with the product. Most features are visible here first.
2. **Backend Development** — Next, learn the API layer, database schemas, and how data flows between frontend and storage.
3. **AI/Agent Development** — Finally, tackle the AI agent system, which depends on understanding both frontend triggers and backend data sources.

Each path is self-contained and can be done independently if you're only working in one area.

## How to Use

1. Read the `AGENTS.md` file at the project root for the master reference
2. Choose your learning path based on your role
3. Work through each module sequentially
4. Complete the practice exercise before moving to the next module
5. After finishing all modules, pick a starter issue from the project board
