## 5. Key Insights

- **Zero price is not the primary value — trust is.** Privacy, open source, and offline capability matter more than cost.
- **Students discover tools via friends, not ads.** College ambassador program is the primary growth lever.
- **Academic calendar is the natural entry point.** Courses → tasks → goals → habits is the adoption funnel.
- **Mobile matters less than expected for Indian students.** Primary device is laptop, but PWA covers mobile.
- **AI is currently in hype cycle — durable value is algorithmic.** Every AI feature must work without AI.
