# Features Specification

## Dashboard & Morning Briefing

### Daily Briefing (7 AM)
- Top 3 tasks prioritized by deadline and energy level
- Overnight opportunities found by radar
- Course target for the day
- Roadmap status overview
- ARIA top pick - single best recommendation

### Productivity Score
- Live 0-100 score
- Factors: tasks completed, study time, sleep quality, streaks
- Updates in real-time

### Activity Heatmap
- GitHub-style visualization
- 6 months of data
- Color intensity based on productivity

### Quick Capture
- One button anywhere in app
- Instantly log task, idea, or resource
- Minimal friction

### Today's Schedule
- Time-blocked view
- Generated by ARIA each morning
- Shows current task with timer

## Task Manager

### Smart Creation
- Type task description
- AI automatically assigns: priority, category, estimated time

### Auto-Reschedule
- Background cron every 15 minutes
- Detects overdue tasks
- Moves to next available slot

### Missed Task Policy
- Zero missed policy
- Every task: done, rescheduled, or explicitly dropped
- No silent failures

### Subtask AI Breakdown
- Ask ARIA to break any complex task
- Creates manageable subtasks automatically

### Dependencies
- Link tasks that must happen in sequence
- Visual dependency graph

### Recurring Tasks
- Daily, weekly, or custom frequency
- Smart rescheduling based on completion

### Sleep-Aware Priority
- Tasks reprioritized based on sleep score each morning
- Low sleep = lighter tasks prioritized

### Escalation
- Push notification after deadline
- Email after 30 minutes
- SMS after 1 hour for critical items

## Opportunity Radar

### Runs Daily at 6 AM
- Background agent scans while you sleep
- Results in morning briefing

### Scans 8 Categories
1. Internships (LinkedIn, Internshala, AngelList)
2. Hackathons (Devfolio, MLH, HackerEarth)
3. Open Source (GitHub Good First Issues, GSoC)
4. Startup Competitions (NASSCOM, Startup India)
5. Fellowships (Buddy4Study, Vidyasaarathi)
6. Freelance (Fiverr, Upwork demand)

### Matching Algorithm
- Skill match score (40% minimum)
- Deadline urgency
- Effort estimation
- Personalization from history

### Critical Alerts
- Opportunity closing in under 48 hours
- Immediate push notification

### Learns from You
- After 2 months, knows which types you act on
- Deprioritizes what you skip

## ARIA - Your AI Agent

### What ARIA Knows
- Every course enrolled and progress
- Every skill (have, learning, gaps)
- All active goals and progress
- Every project phase and last action
- All income sources and trends
- Daily task completion rate
- Every idea captured
- Sleep patterns and productivity correlation
- Every opportunity shown
- Preferences and patterns

### What ARIA Does
- Chat with you naturally
- Answer questions about your data
- Take actions through conversation
- Generate daily briefings
- Create weekly reviews
- Suggest what to focus on
- Break down complex tasks

### 8 AI Sub-Agents
1. Daily Briefing Agent
2. Missed Task Checker
3. Opportunity Radar
4. Roadmap Update Checker
5. Weekly Review Agent
6. Bedtime and Sleep Agent
7. Habit Miss Checker
8. Course Progress Nudge

### Communication
- Orchestrator receives message first
- Calls relevant agents in parallel
- Combines outputs into coherent reply
- All data flows through Supabase
