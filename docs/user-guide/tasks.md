## Creating Tasks

You can create tasks in two ways:

**Quick Add** -- Use the input bar at the top of the dashboard or Tasks page. Type the task name and press Enter. It saves instantly with default settings (Medium priority, no due date).

**Full Form** -- Click the **Add Task** button to open the full form. This lets you set:

| Field | Description |
|---|---|
| Title | The task name (required) |
| Description | Optional notes or details |
| Priority | Low, Medium, or High |
| Due Date | Optional deadline |
| Category | Group tasks by category (study, work, personal, etc.) |
| Estimated Minutes | How long you think it will take |
| Recurring | Set up repeating tasks (daily, weekly, monthly) |
| Link to Goal | Connect this task to a larger goal |
| Link to Project | Connect this task to a project phase |

## Organizing with Priorities and Due Dates

Tasks are sorted by priority on your task list. High-priority tasks appear at the top.

- **High Priority** -- Due soon or critical. Shown with a red indicator.
- **Medium Priority** -- Normal importance. Shown with a yellow indicator.
- **Low Priority** -- Nice-to-do. Shown with a gray indicator.

Set a due date and overdue tasks will be flagged automatically.

## Task Statuses

Every task moves through these statuses:

| Status | Meaning |
|---|---|
| **Pending** | Created but not started |
| **In Progress** | You're actively working on it |
| **Completed** | Marked done |
| **Missed** | Was overdue and never completed |

> Change a task's status by clicking its current status badge and selecting a new one from the dropdown.

## Task Dependencies

You can link a task to another task that must be completed first. For example, "Review lecture notes" might depend on "Attend lecture."

When you set a dependency, ARIA will remind you to complete the prerequisite task first.

## Completing Tasks

Click the checkbox next to a task to mark it **Completed**. The task gets a completion timestamp and moves to your completed list. You can still view completed tasks by switching the filter.

> Completed tasks are automatically archived after 30 days but remain accessible in your history.

## Task Views

The Tasks page offers these views:

- **List View** -- Default. All tasks in a scrollable list with priority colors.
- **Calendar View** -- Tasks plotted on a calendar by due date. Shows your month at a glance.

Switch between views using the toggle at the top of the Tasks page.
