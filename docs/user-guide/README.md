Welcome to the ARIA OS user guide. This is your central resource for learning how to use the Second Brain OS -- your personal AI-powered productivity system.

## What is ARIA OS?

ARIA OS (Adaptive Reasoning Intelligent Assistant Operating System) is a personal productivity platform built for BTech CSE students. It combines task management, habit tracking, sleep logging, goal planning, project management, and an AI assistant into one unified dashboard.
